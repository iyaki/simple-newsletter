Simple Newsletter API

/subscriptions/

Query Params

email
Email to send the subscribed feed news
uri
URI from the Atom/RSS Feed to subscribe

/subscriptions/confirmation

Query Params

email
Email to send the subscribed feed news
uri
URI from the Atom/RSS Feed to subscribe
token
Confirmation token

/subscriptions/cancellation

Query Params

email
Email asociated with the subscription
uri
URI of the subscripted feed
token
Authorization token
Known limitations
  • Currently the service only sends up to 3 newsletters by day from a single subscription (Feed/E-mail combination).
  • For performance reasons all the subscriptors to a Feed are BCCed on the same e-mail. If you don't find the newsletter e-mails in your inbox please check your spam folder and mark the message as "Not Spam".
